🚀 Cómo montar el motor en Render — paso a paso
✅ Render es gratis, permanente y sin VPN desde Cuba.
PASO 1 — Sube los archivos del servidor a GitHub
Asegúrate de tener en tu repositorio estos 3 archivos en la raíz:
qcol_server.py   requirements.txt   render.yaml
PASO 2 — Crea el servicio en Render
1. Ve a dashboard.render.com
2. Clic en New + → Web Service
3. Conecta tu repo: laffitacamargof-create/quantum-computing-open-lab
4. Render detecta el render.yaml automáticamente
5. Plan: Free → Clic Create Web Service
PASO 3 — Esperar el deploy
Render instala Qiskit + Cirq — tarda 5-8 minutos la primera vez.
Cuando diga Deploy succeeded tu URL está lista.
PASO 4 — Copiar la URL y pegar arriba
En Render verás tu URL fija. Ejemplo:
https://qcol-motor.onrender.com
Pégala en el campo de arriba y presiona PROBAR.
PASO 5 — Mantenerlo activo (recomendado)
Render Free se duerme tras 15 min sin uso. Para evitarlo:
1. Ve a uptimerobot.com → cuenta gratis
2. New Monitor → HTTP(s) → URL: [tu-url]/health
3. Intervalo: 5 minutos → Guardar
El motor nunca se duerme.

🌐 Publicar en Internet — GitHub Pages
✅ GitHub Pages es la forma más fácil. Tu ecosistema queda en: laffitacamargof-create.github.io/quantum-computing-open-lab
PASO 1 — Subir archivos a GitHub
Sube estos archivos a tu repositorio: index.html, qcol-founder.html, y todos los módulos que hayas descargado.
PASO 2 — Activar GitHub Pages
En tu repositorio: Settings → Pages → Source: "Deploy from a branch" → Branch: main → Save
PASO 3 — Motor Render
Ve a la pestaña ⚙️ Motor Render → sigue las instrucciones → pega tu URL de Render.
PASO 4 — Configurar IA (opcional)
Crea cuenta en Groq → Obtén API key gratuita → Configura en pestaña IA

Tu ecosistema está en internet. Comparte la URL con cualquiera.
