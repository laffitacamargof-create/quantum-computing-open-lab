❓ AYUDA — QCOL Quantum Computing Open Lab
🌐 USO GENERAL
🚀 ¿Cómo uso el ecosistema?
Abre index.html en cualquier navegador. Verás el menú principal con todos los módulos disponibles. No necesitas instalar nada.
👤 ¿Cómo creo una cuenta?
Clic en ENTRAR → Crear cuenta. Llena tu nombre, usuario y contraseña. Tu cuenta guarda tus circuitos creados en el Arquitecto.
👑 ¿Qué es la Cuenta Fundador?
Es el panel de control total del ecosistema. Permite configurar el motor de ejecución, la IA, GitHub, gestionar usuarios y agregar/eliminar contenido. Accede con la contraseña del fundador.
⚙️ ARQUITECTO CUÁNTICO
⚙️ ¿Qué puede hacer el Arquitecto?
Escribir y ejecutar código cuántico en Qiskit, Cirq, PennyLane o QuTiP. Visualizar estados cuánticos en la esfera de Bloch. Guardar circuitos en tu cuenta. Descargar código como .py.
🌐 ¿Cómo funciona la esfera de Bloch?
Visualiza el estado de un qubit:
• Polo norte = |0⟩ (θ=0°)
• Polo sur = |1⟩ (θ=180°)
• Ecuador = superposición
• φ = fase relativa
Los botones H, X, Y, Z aplican puertas y mueven el vector.
💾 ¿Cómo guardo mis circuitos?
Pestaña 💾 GUARDAR en el Arquitecto → escribe un nombre → Guardar en cuenta. Necesitas sesión activa. Tus circuitos aparecen en Mi Cuenta.
📂 PROYECTOS Y LIBRERÍA
📂 ¿Cómo ejecuto un proyecto?
Ve a Proyectos → encuentra el notebook → clic en ▶ EJECUTAR. Se abrirá en el Arquitecto donde puedes ejecutarlo y modificarlo (sin guardar los cambios en el original).
📚 ¿Cómo agrego contenido a la Librería?
Solo administradores y fundador pueden agregar contenido. Entrá como administrador → aparece la barra dorada → + AGREGAR RECURSO. Puedes agregar PDFs, videos o notebooks.
🔧 ADMINISTRACIÓN
🔧 ¿Cómo activo el motor de ejecución real?
Panel Fundador → pestaña 🚀 Motor → sigue las instrucciones para configurar el motor → pega la URL → PROBAR. Cuando el motor esté activo verás ● MOTOR en la barra del Arquitecto.
🤖 ¿Cómo activo la IA real?
Panel Fundador → pestaña 🤖 IA → elige proveedor → pega tu API key → Guardar. Groq es gratuito y funciona sin restricciones geográficas.
🧩 ¿Cómo actualizo un módulo?
Cada módulo es un archivo HTML independiente. Para actualizarlo: descarga el nuevo archivo del módulo → súbelo a GitHub reemplazando el antiguo. El ecosistema lo carga automáticamente sin tocar los demás módulos.
➕ ¿Cómo creo un módulo nuevo?
Panel Fundador → pestaña 🧩 Módulos → sección Agregar módulo. Define el nombre, el archivo HTML y el icono. El módulo aparecerá en el menú principal automáticamente cuando subas el archivo a GitHub.
